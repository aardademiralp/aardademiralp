# Selam, ben Arda.

İstinye Üniversitesi **Bilişim Güvenliği Teknolojisi** programında 1. sınıf öğrencisiyim. 22 yaşındayım ve özellikle ağ güvenliği, sızma testleri ve sistem sıkılaştırma konularına odaklanıyorum.

---

### 🛠️ Teknolojiler ve Araçlar

* **İşletim Sistemleri:** Kali Linux, Windows Server
* **Diller:** Rust (Güvenlik araçları geliştirme), Python, Bash
* **Araçlar:** Nmap, Wireshark, Metasploit, Bettercap, VirtualBox
* **İlgi Alanları:** Ağ Analizi, SSL/TLS Denetimi, Otomasyon

### 🛡️ Öne Çıkan Projeler

* **Network Security Labs** - İstinye Üniversitesi'ndeki çalışmalarım sırasında hazırladığım dokümantasyon ve araçlar koleksiyonu.
